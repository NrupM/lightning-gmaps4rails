## <img src="https://www.seeklogo.net/wp-content/uploads/2016/07/Ruby-logo.png" height="20px"> Lightning talk on Ruby gem `gmaps4rails`

* Steps to get started

1. git clone repo
2. `$ bundle install `
3. `$ rails db:create`
4. `$ rails s`
5. open your browser and go to localhost:3000
